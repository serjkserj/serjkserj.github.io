# CHECKLIST TEMPLATE

## Название чек-листа

### 1. Общая информация
- [ ] Понятна цель проверки
- [ ] Понятна стадия проекта
- [ ] Понятен раздел / модель

### 2. Геометрия
- [ ] Проверена базовая целостность модели
- [ ] Проверены уровни и оси
- [ ] Проверены критичные элементы

### 3. Данные
- [ ] Проверены обязательные параметры
- [ ] Проверены ключевые коды / классификация
- [ ] Проверены зоны и площади

### 4. IFC
- [ ] Проверена логика экспорта
- [ ] Проверено содержимое IFC
- [ ] Проверены критичные ошибки после экспорта

### 5. Итог
- [ ] Сформулированы замечания
- [ ] Определены критичные риски
- [ ] Понятно, что делать дальше

## Как использовать шаблон

- Сначала укажи цель проверки и стадию проекта.
- Не проверяй все одинаково глубоко на любом этапе.
- Отделяй блокеры от второстепенных замечаний.
- После чек-листа должен оставаться понятный следующий шаг.
