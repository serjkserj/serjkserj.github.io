# CHECKLIST PREPARE EXPERTISE

## 1. Структура и читаемость модели

- [ ] Модель читается как система, а не как авторский файл
- [ ] Основные сущности распознаются однозначно
- [ ] Нет критичного структурного хаоса

## 2. Негеометрический слой

- [ ] Ключевые параметры заполнены
- [ ] Классификация и атрибуты не противоречат друг другу
- [ ] Пространства, зоны и показатели согласованы

## 3. Междисциплинарная логика

- [ ] Известные междисциплинарные конфликты разобраны
- [ ] Понятно, где риск локальный, а где системный
- [ ] Команда не проверяет архитектурную модель в отрыве от смежников

## 4. Финальная готовность

- [ ] Модель готова к более формальному чтению
- [ ] Повторяющиеся слабые места учтены заранее
- [ ] Команда понимает реальные риски перед передачей

Этот чек-лист не должен заменять чек-лист к АГР: его полезно использовать именно как отдельную рамку для более строгого внешнего чтения модели.
