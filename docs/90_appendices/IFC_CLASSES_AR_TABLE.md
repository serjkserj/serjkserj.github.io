# IFC CLASSES AR TABLE

| IFC-класс | Простое объяснение | Где важен для АР |
|---|---|---|
| IfcProject | Верхний контекст проекта | Общая структура обменной модели |
| IfcSite | Участок / площадка | Контекст объекта и часть внешней логики чтения |
| IfcBuilding | Здание как целое | Общая структурная опора модели |
| IfcBuildingStorey | Этаж | Этажность, уровни, пространственная структура |
| IfcWall | Стена | Базовая архитектурная сущность |
| IfcSlab | Перекрытие / плита | Этажная логика и конструктивно-архитектурная читаемость |
| IfcDoor | Дверь | Архитектурная структура и проверка элементов |
| IfcWindow | Окно | Фасады и архитектурное чтение |
| IfcSpace | Пространство / помещение | Зоны, площади, ТЭП, проверка расчетной логики |
