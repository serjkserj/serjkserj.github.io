# MODEL ELEMENT PARAMETER SOURCE TABLE

| Элемент / сущность | Обязательный тип данных | Типичный источник значения | Кто чаще отвечает |
|---|---|---|---|
| Помещение / IfcSpace | Наименование, назначение, часть расчетной логики | Архитектурное решение, таблицы помещений, логика проекта | Архитектор / BIM-координатор контролирует |
| Зона | Тип зоны, расчетные признаки, показатели | Расчетная логика проекта, таблицы, требования к АГР | Архитектор / BIM-координатор |
| Участок | Базовые идентификационные и площадные данные | Исходные документы, расчеты, проектная ситуация | Проектировщик / координатор контролирует |
| Архитектурный элемент | Классификация, базовые свойства, роль в модели | Проектное решение, карта параметров, классификатор | Архитектор / BIM-координатор |
| IFC-атрибут | Значение для обмена и проверки | Параметры модели, правила экспорта, карта соответствия | BIM-координатор / BIM-специалист |
| Показатель / ТЭП | Расчетное значение и его источник | Зоны, площади, таблицы расчета, методика | Архитектор / BIM-координатор / расчетная логика проекта |

Примечание:
- внутренние поля бюро и `ATR*`-параметры не должны автоматически попадать в это базовое ядро как универсальная норма.
