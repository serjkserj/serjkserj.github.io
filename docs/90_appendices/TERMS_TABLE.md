# TERMS TABLE

Таблица ниже смешивает два слоя: устойчивые общие термины `BIM` и локально source-sensitive термины текущего document-layer. Там, где термин зависит от действующих требований, читать его нужно вместе с карточками документов и профильными модулями, а не как автономную норму.[^appendix-terms-table]

| Термин | Расшифровка | Простое объяснение | Где встречается |
|---|---|---|---|
| BIM | Building Information Modeling / информационное моделирование | Информационное моделирование объекта | Во всей логике проекта |
| ЦИМ | Цифровая информационная модель | Цифровая модель объекта как носитель геометрии и данных | Нормативные и проектные документы |
| ТСИМ | Требуемая средой цифровая информационная модель | Цифровая модель для конкретного внешнего рубежа передачи | Экспертиза, внешняя передача |
| ЦИМ АГР | Цифровая информационная модель для согласования АГР | Цифровая модель для согласования АГР | АГР, IFC-пакет, XML |
| НПМ | Низкополигональная модель | Упрощенная полигональная 3D-модель для раннего или базового визуального представления | АГР, визуальный пакет |
| ВПМ | Высокополигональная модель | Более детализированная полигональная 3D-модель для визуального представления фасадов и архитектурных деталей | АГР, визуальный пакет |
| IFC | Industry Foundation Classes | Формат обмена BIM-данными | Экспорт, проверка, АГР, экспертиза |
| IFC4 ReferenceView | Reference View для IFC4 | Тип модельного вида IFC для внешней передачи | АГР, внешняя выдача |
| IFC SPF | STEP Physical File | Текстовый файловый формат IFC | Экспорт IFC |
| LandXML | Land eXtensible Markup Language | Формат обмена инженерной цифровой моделью местности | Изыскания, ИЦММ, внешняя передача |
| АГР | Архитектурно-градостроительное решение | Согласование архитектурного облика | Согласование архитектурного облика |
| Экспертиза | Экспертиза проектной документации и результатов инженерных изысканий | Проверка проектной документации и модели | ПД, требования, замечания |
| ТЭП | Технико-экономические показатели | Показатели объекта: площади, объемы и др. | XML, проверки, расчеты |
| СПП в ГНС | Суммарная поэтажная площадь в габаритах наружных стен | Один из расчетных показателей площади | АГР, расчеты, зоны |
| АР | Архитектурные решения | Архитектурный раздел и архитектурная модель | Архитектурная модель |
| КР | Конструктивные решения | Конструктивный раздел | Смежный раздел |
| МССК | Московская строительная система классификаторов | Классификатор элементов и сущностей | Параметры, IFC, проверки |
| EIR | Exchange Information Requirements | Требования к информации и поставке модели | BIM-процесс, документы, требования |
| BEP | BIM Execution Plan | План, как команда будет реализовывать BIM-требования | BIM-процесс, организация проекта |
| ПИМ | План информационного моделирования | Русскоязычный рабочий аналог или локальная форма BEP | BIM-процесс, организация проекта, проектные инструкции |
| MIDP | Master Information Delivery Plan | План поставки информации по срокам и участникам | BIM-процесс, контрольные точки |
| CDE | Common Data Environment | Среда общих данных | Обмен файлами, процесс, координация |
| LOD | Level of Development | Уровень графической проработки | Зрелость модели |
| LoIN | Level of Information Need | Уровень потребности в информации | Требования к модели и комплекту поставки |
| BCF | BIM Collaboration Format | Формат аннотаций и замечаний, связанных с моделью | Координация, замечания, АГР |
| QC | Quality Control | Контроль качества модели | Проверки, аудит, подготовка к выдаче |
| IfcSpace | IFC-класс пространств | Пространство / помещение в IFC | Зоны, площади, ТЭП, IFC |
| IfcWall | IFC-класс стен | Стена как сущность IFC | IFC-классы, атрибуты, проверки |
| IfcSite | IFC-класс участка | Участок как сущность IFC | АГР, участок, площадные данные |
| Pset | Property Set | Стандартный набор свойств IFC | Атрибуты, IFC, экспертиза |
| RusSet | Russian Set / национальный набор свойств | Набор атрибутов для внешнего цифрового контура | АГР, экспертиза, IFC |
| RUS_* | Префикс внешних полей набора атрибутов | Конкретное внешнее поле из набора параметров | IFC, карта параметров, проверка |
| Классификация элемента | Классификация элемента | Отнесение элемента к сущности, коду или классу | IFC, параметры, экспертиза |
| Идентификация элемента | Идентификация элемента | Устойчивое имя, номер или связь элемента с частью объекта | IFC, передача, проверка |
| Юридическая значимость | Юридическая значимость цифрового материала | Официальный статус цифрового файла как передаваемого материала | Внешняя передача, АГР, экспертиза |
| Обязательное требование | Обязательное требование | То, что должно быть выполнено по документу или официальному локальному контуру | Документы, приложения, чек-листы, замечания |
| Методика | Методика | Способ расчета, чтения документа или выполнения проверки | ТЭП, проверки, рабочие таблицы |
| Локальная практика | Локальная практика | Рабочая схема конкретного проекта или организации | Проектные инструкции, внутренние правила |
| Внутренняя служебная надстройка | Внутренняя служебная надстройка | Вспомогательные поля и признаки команды, не равные внешней норме | Revit-параметры, внутренние таблицы, ATR* |
| Пакет передачи | Пакет передачи | Управляемый комплект файлов, приложений и сопровождающих материалов | IFC, XML, BCF, АГР, экспертиза |
| Внешний рубеж | Внешний рубеж | Формальный этап, где модель должна быть прочитана извне | IFC-обмен, АГР, экспертиза, цифровая выдача |
| Аннотация | Аннотация | Управляемый сопровождающий слой пояснений и замечаний к модели | BCF, АГР, замечания |
