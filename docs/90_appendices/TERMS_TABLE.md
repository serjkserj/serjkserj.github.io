# TERMS TABLE

| Термин | Простое объяснение | Где встречается |
|---|---|---|
| BIM | Информационное моделирование объекта | Во всей логике проекта |
| ЦИМ | Цифровая информационная модель | Нормативные и проектные документы |
| ТСИМ | Цифровая модель для конкретного внешнего рубежа передачи | Экспертиза, внешняя передача |
| ЦИМ АГР | Цифровая модель для согласования АГР | АГР, IFC-пакет, XML |
| IFC | Формат обмена BIM-данными | Экспорт, проверка, АГР, экспертиза |
| IFC4 ReferenceView | Тип модельного вида IFC для внешней передачи | АГР, внешняя выдача |
| IFC SPF | Текстовый файловый формат IFC | Экспорт IFC |
| LandXML | Формат обмена инженерной цифровой моделью местности | Изыскания, ИЦММ, внешняя передача |
| АГР | Архитектурно-градостроительное решение | Согласование архитектурного облика |
| Экспертиза | Проверка проектной документации и модели | ПД, требования, замечания |
| ТЭП | Показатели объекта: площади, объемы и др. | XML, проверки, расчеты |
| СПП в ГНС | Площадь в габаритах наружных стен | АГР, расчеты, зоны |
| АР | Архитектурные решения | Архитектурная модель |
| КР | Конструктивные решения | Смежный раздел |
| МССК | Классификатор элементов и сущностей | Параметры, IFC, проверки |
| EIR | Требования к информации и поставке модели | BIM-процесс, документы, требования |
| BEP | План, как команда будет реализовывать BIM-требования | BIM-процесс, организация проекта |
| MIDP | План поставки информации по срокам и участникам | BIM-процесс, контрольные точки |
| CDE | Среда общих данных | Обмен файлами, процесс, координация |
| LOD | Уровень графической проработки | Зрелость модели |
| LoIN | Уровень потребности в информации | Требования к модели и комплекту поставки |
| BCF | Формат аннотаций и замечаний, связанных с моделью | Координация, замечания, АГР |
| IfcSpace | Пространство / помещение в IFC | Зоны, площади, ТЭП, IFC |
| IfcWall | Стена как сущность IFC | IFC-классы, атрибуты, проверки |
| IfcSite | Участок как сущность IFC | АГР, участок, площадные данные |
| Pset | Стандартный набор свойств IFC | Атрибуты, IFC, экспертиза |
| RusSet | Набор атрибутов для внешнего цифрового контура | АГР, экспертиза, IFC |
| RUS_* | Конкретное внешнее поле из набора параметров | IFC, карта параметров, проверка |
| Классификация элемента | Отнесение элемента к сущности, коду или классу | IFC, параметры, экспертиза |
| Идентификация элемента | Устойчивое имя, номер или связь элемента с частью объекта | IFC, передача, проверка |
| Юридическая значимость | Официальный статус цифрового файла как передаваемого материала | Внешняя передача, АГР, экспертиза |
| Обязательное требование | То, что должно быть выполнено по документу или официальному локальному контуру | Документы, приложения, чек-листы, замечания |
| Методика | Способ расчета, чтения документа или выполнения проверки | ТЭП, проверки, рабочие таблицы |
| Локальная практика | Рабочая схема конкретного проекта или организации | Проектные инструкции, внутренние правила |
| Внутренняя служебная надстройка | Вспомогательные поля и признаки команды, не равные внешней норме | Revit-параметры, внутренние таблицы, ATR* |
| Пакет передачи | Управляемый комплект файлов, приложений и сопровождающих материалов | IFC, XML, BCF, АГР, экспертиза |
| Внешний рубеж | Формальный этап, где модель должна быть прочитана извне | IFC-обмен, АГР, экспертиза, цифровая выдача |
| Аннотация | Управляемый сопровождающий слой пояснений и замечаний к модели | BCF, АГР, замечания |
| QC | Контроль качества модели | Проверки, аудит, подготовка к выдаче |
