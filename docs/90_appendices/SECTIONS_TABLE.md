# SECTIONS TABLE

| Раздел / тема | Простое объяснение | Почему важно BIM-координатору АР |
|---|---|---|
| АР | Архитектурные решения | Центральная дисциплина для архитектурной модели |
| КР | Конструктивные решения | Влияет на междисциплинарную координацию и коллизии |
| ИОС | Инженерные системы и оборудование | Влияют на координацию и читаемость общей модели |
| СПОЗУ / генплан | Участок, посадка, благоустройство и общая проектная ситуация | Важны для контекста объекта и части АГР-логики |
| ТЭП | Показатели объекта и его площадей | Влияют на проверку, XML и внешнюю процедуру |
| IFC | Обменное представление модели | Нужно для передачи, чтения и внешних проверок |
| Замечания | Формализованные сигналы о проблемах модели или процесса | Помогают превращать проверку в действие |
