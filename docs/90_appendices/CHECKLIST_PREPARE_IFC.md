# CHECKLIST PREPARE IFC

## 1. Цель экспорта

- [ ] Понятно, зачем нужен текущий IFC
- [ ] Понятно, кто будет читать файл дальше
- [ ] Понятна глубина текущей проверки

## 2. Подготовка модели

- [ ] Структура модели читается последовательно
- [ ] Критичные уровни, пространства и элементы проверены
- [ ] Ключевые данные заполнены для текущей задачи

## 3. Проверка перед экспортом

- [ ] Устранены известные критичные блокеры
- [ ] Понятны оставшиеся риски
- [ ] Экспорт не используется как первая серьезная проверка

## 4. Проверка результата

- [ ] IFC открылся и читается как обменная модель
- [ ] Ключевые сущности присутствуют и распознаются
- [ ] Зафиксированы проблемы, которые проявились именно после экспорта

Этот чек-лист полезно проходить вместе с модулем про `IFC` и главой про предэкспортную проверку, чтобы экспорт не становился первой настоящей диагностикой модели.
