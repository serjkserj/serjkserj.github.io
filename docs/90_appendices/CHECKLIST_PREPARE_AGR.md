# CHECKLIST PREPARE AGR

## 1. Архитектурная логика

- [ ] Уровни, этажность и основные элементы собраны последовательно
- [ ] Фасадная и пространственная логика читается без явного хаоса
- [ ] Модель не держится на случайных обходных решениях

## 2. Зоны, площади, показатели

- [ ] Зоны и помещения соответствуют задаче проекта
- [ ] Площади и ТЭП можно объяснить по источнику
- [ ] Расчетная логика не отделена от модели искусственно

## 3. Данные и IFC

- [ ] Ключевые атрибуты для текущего этапа заполнены
- [ ] Понятна цель IFC-подготовки
- [ ] Критичные проблемы обменного слоя проверены заранее

## 4. Комплект подачи

- [ ] Модель и сопровождающие материалы согласованы
- [ ] Команда понимает состав пакета
- [ ] Известные риски проговорены до подачи

Этот чек-лист особенно полезно проходить вместе с модулем про АГР и модулем про `QC`, чтобы не сводить готовность к подаче только к формальному наличию файлов.
