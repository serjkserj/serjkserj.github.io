# Модуль 0. Базовая картина проектирования зданий для новичка

## Зачем нужен этот модуль

Этот модуль нужен для того, чтобы у читателя появилась карта местности.

Пока этой карты нет, термины вроде “АГР”, “ПД”, “IFC”, “экспертиза”, “СПП”, “АР” и “КР” воспринимаются как разрозненные слова без системы. Именно поэтому у многих новичков и даже у работающих специалистов возникает ощущение, что в голове каша.

## Главная задача модуля

Задача этого модуля — показать, как устроена среда проектирования зданий:

- какие бывают стадии;
- какие бывают разделы;
- кто участвует в проекте;
- что такое площади, зоны и ТЭП;
- зачем нужны АГР и экспертиза;
- где в этой системе появляется BIM-координатор.

## Что будет дальше

После этого модуля читатель должен перестать воспринимать тему как набор отдельных документов и требований. Вместо этого у него должна появиться цельная логика:

```text
проект → стадии → разделы → модели → IFC → проверки → АГР / экспертиза
```
